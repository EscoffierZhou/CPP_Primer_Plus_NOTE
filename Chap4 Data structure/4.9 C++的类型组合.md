# 4.9 C++的类型组合

本章介绍了数组/结构/指针,可以以各种方式组合他们

```cpp
struct antarctica_years_end
{
	int year;
};
antarctica_years_end s01,s02,s03; // 实例化对象

s01.year = 1998l // 成员运算符赋值
    
antarctica_years_end *pa = &s02;  // 初始化指针 
pa->year = 1999; // 结构运算符赋值

antarctica_years_end trio[3];
trio[0].year = 2003; // 结构体指针初始化
(trio + 1)->year = 2004;

const antarctica_years_end *arp[3] = {&s01,&s02,&s03}; // 指针数组
cout << arp[1]->year << endl; // 成员运算符
const antarctica_years_end ** ppa = arp; // 指针数组中的指针的指针,或者直接auto
auto ppb = arp;
cout << (*ppa)->year << endl;
cout << (*(ppb+1))->year << endl;
```
