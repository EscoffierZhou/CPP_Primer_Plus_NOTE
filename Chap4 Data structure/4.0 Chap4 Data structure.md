# Chap4 Data structure

## 本章内容

1.创建和使用数组。

2.创建和使用C-风格字符串。

3.创建和使用string类字符串。

4.使用方法`getline( )`和`get( )`读取字符串。

5.混合输入字符串和数字。

6.创建和使用结构。

7.创建和使用共用体。

8.创建和使用枚举。

9.创建和使用指针。

10.使用new和delete管理动态内存。

11.创建动态数组。

12.创建动态结构。

13.自动存储、静态存储和动态存储。

14.vector和array类简介。
